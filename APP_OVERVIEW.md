# 土地家屋調査士 学習アプリ — 構成概要

## アプリ概要

**URL**: https://three100.github.io/chosashi-app/  
**リポジトリ**: https://github.com/ThREE100/chosashi-app  
**目的**: 土地家屋調査士試験（国家資格）の過去問演習・暗記学習・書式問題レビューをPWA（Progressive Web App）として提供する。

---

## 技術スタック

| 項目 | 内容 |
|------|------|
| フレームワーク | Vite + React + TypeScript |
| スタイリング | Tailwind CSS v4 |
| PWA | vite-plugin-pwa（Workbox）|
| 認証・DB | Supabase（Magic Link認証＋PostgreSQL）|
| デプロイ | GitHub Pages（GitHub Actions自動デプロイ）|
| ローカル保存 | localStorage（高速・オフライン対応）|
| クロスデバイス同期 | Supabase（PC ↔ iPhone 自動同期）|

## 主要機能

### 学習機能
| 機能 | 説明 |
|------|------|
| 年度別演習 | 指定年度の択一問題20問を通しで解く |
| 科目別演習 | 民法・不動産登記法・土地家屋調査士法で絞り込み |
| 苦手をまとめて復習 | 直近で間違えた全問題を一括演習 |
| 今日の復習 | SRS（間隔反復）で本日が復習日の問題のみ出題 |
| 暗記○×モード | 一問一答形式でOXカードを演習 |
| 用語フラッシュカード | 重要用語の表裏カード学習 |
| 書式レビュー | 書式問題を閲覧・自己採点するレビューUI |

### 記録・分析機能
| 機能 | 説明 |
|------|------|
| 解答記録 | 各問題の正解数・不正解数・直近結果を記録 |
| 全体統計 | 解答済み問題数・延べ正答率 |
| 科目別統計 | 科目ごとの正答率・苦手度 |
| 年度別進捗 | 各年度の回答済み問題数・正答率 |

### 間隔反復（SRS: Spaced Repetition System）
- SM-2アルゴリズム（簡易版）を実装
- 間違えた問題を翌日→3日後→約1週間後…と自動スケジュール
- 初回正解した問題は「習得済み」として追跡しない
- 「今日の復習」バナーで本日分の復習問題数を表示

### クロスデバイス同期
- Supabase（PostgreSQL）にログインした全端末で記録を共有
- ログイン方式: Magic Link（メールアドレスにリンク送付）
- ログイン時に双方向同期（リモート→ローカル、ローカル→リモート）
- 認証前に積み上げた記録も初回ログイン時に自動アップロード

```
src/data/

├── Home.tsx           # ホーム画面（タブ切替・統計表示・ナビゲーション）

├── KijutsuReview.tsx  # 書式レビュー画面
```

## Supabaseテーブル構成

| テーブル名 | 用途 |
|-----------|------|
| `progress` | 択一・暗記の解答記録（question_id, correct, wrong, last_result） |
| `srs` | 間隔反復スケジュール（question_id, ease, ivl, reps, due） |

全テーブルにRLS（Row Level Security）を適用し、自分のデータのみアクセス可能。

## 対象ユーザー

土地家屋調査士試験の受験生。特に：

- スマートフォン（iPhone）とPC両方で学習記録を共有したい方
- 苦手問題を効率よく反復学習したい方（SRS機能）
