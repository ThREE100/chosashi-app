# 新しい問題の記事を生成するためのプロンプトテンプレート

新しい年度・問題番号の解説記事を作りたいときは、Claude Codeに以下のプロンプトをそのまま（または年度・問題番号を書き換えて）投げてください。このリポジトリ（`ThREE100/chosashi-app`）を開いた状態、または `note-articles/format-template.md` の内容をあらかじめ読み込ませた状態で実行するとスムーズです。

---

## プロンプト例

```
土地家屋調査士試験 {令和◯年度} {午前/午後}の部 第{n}問について、
note-articles/format-template.md の執筆ルールに従って解説記事を作成してください。

手順：
1. まだ問題文・公式正答を確認していない場合は、法務省の公式サイト
   (https://www.moj.go.jp/MINJI/minji05_00732.html 等、年度に応じて検索)
   から試験問題原本と正答資料のPDFを取得し、問題文・正解番号を確認してください。
1.5. 土地家屋調査士試験は同じ問題(ほぼ同一の問題文)が別の年度で再出題される
   ことがあります。note-articles/ 配下の既存の年度フォルダ(作成済みの各
   r{年度}-mondai/)の問題文と照合し、実質的に同一の問題が既に解説済みで
   ないか確認してください。該当する場合は、出典行の直後に
   `※この問題は{既に解説済みの年度}度で解説済みです` の一行を追加して
   ください。
2. 各肢(ア〜オ等)について、条文(e-Gov法令検索)や実務解説サイトで
   正誤の根拠を確認してください。推測で断定せず、確認できなかった点は
   確認事項ブロックに正直に記載してください。
3. format-template.md のテンプレート構造(1〜10)に沿って記事を作成してください。
4. 完成した記事を note-articles/r7-mondai/(またはその年度用のフォルダ)/q{n}-{内容の略称}.md
   として保存してください。
5. 最後に、見出し画像用フレーズを5つ程度、語り口調で作成してください。
```

## 年度が変わる場合

令和7年度以外の年度を扱う場合は、`note-articles/` 直下に `r{年度}-mondai/` という新しいフォルダを作り、同じ命名規則（`q{問題番号}-{内容の略称}.md`）でファイルを保存してください。README.md のフォルダ構成の記載も忘れずに更新してください。

## 記述式問題（第21問・第22問）について

このシリーズは択一式（ア〜オの5肢問題）の解説を前提としたテンプレートです。記述式問題は問題の性質が異なる（座標計算・図面作成・申請書の記入など）ため、そのまま同じフォーマットは使えません。記述式を扱う場合は、まず新しいフォーマットを検討し、`format-template.md` とは別のテンプレートファイルを作成することを推奨します。

## 過去の検証で見つかった注意点（今後の生成でも起こりうる落とし穴）

- ユーザーや資料が問題番号を勘違いしていることがある。必ず法務省の原本と突き合わせて確認する。
- 具体的な金額・年数・条文番号などの数値は、初回の推論だけで断定せず、可能な限り一次資料（条文そのもの、法務局公式サイト）で裏付けを取る。二次資料（ブログ・予備校サイト）だけに頼った場合は、確認事項ブロックにその旨を明記する。
- 正解の選択肢番号（1〜5）は、公式正答PDFの「番号」であって、肢の組合せ（例：アウ）とセットで確認すること。
